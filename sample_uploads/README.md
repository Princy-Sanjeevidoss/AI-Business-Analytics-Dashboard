# Sample Upload Files

Use these files from the dashboard's Data Manager tab.

Recommended order:

1. Import `customers_sample.csv` in Customers.
2. Import `products_sample.csv` in Products.
3. Import `employees_sample.csv` in Employees.
4. Import `sales_sample.csv` in Sales.
5. Import `payments_sample.csv` in Payments.

Sales and Payments use `customer_id` / `product_id`, so import Customers and Products first.
